This plugin will embed Myanmar Unicode Font in your wordpress blog.

== Description ==

In 2011, most the Myanmar popular websites has been change to Unicode. But some of the internet cafes are not ready to add. So, this plugin can help you to make font embed but render can't work perfectly. This plugin also make to see clear RSS. It will add automatically Unicode font tag in RSS. So, subscriber can see clearly if they install some of the unicode font.

* Support Wp touch for iPhone and iPod (include wp touch pro theme )

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Download zip file and extract it.
1. Upload unzip folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. WP Admin -> Settings -> MMUnicode Embed

== Frequently Asked Questions ==

= Can't see on iOS =

* TTF HTML5 font embed only support on iOS 4.0 or later

= Can't see on Android , Chrome  =

* Android and chrome can't support embed font render now. Try to use with firefox on Android.

= Can't see on Mac Firefox  =

* Yes, mac firefox can't render AAT on font embed . You need to change config to see AAT font render. 

about:config on address bar.

downloadable_fonts.sanitize to false
